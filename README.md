# TypeScript Fundamentals Test

This project contains a series of exercises to assess TypeScript fundamentals knowledge.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Check your work:
```bash
npm run check
```

This will compile the TypeScript files and show any errors. All exercises should compile without errors when completed correctly.

## Exercises

### 1. Basic Types (`src/01-basic-types.ts`)
Complete the type annotations for variables and function signatures.

### 2. Interfaces (`src/02-interfaces.ts`)
Define interfaces for objects and implement them correctly.

### 3. Classes (`src/03-classes.ts`)
Create classes with proper TypeScript syntax including access modifiers.

### 4. Functions (`src/04-functions.ts`)
Implement functions with proper type annotations, optional parameters, and overloads.

### 5. Generics (`src/05-generics.ts`)
Create generic functions and classes.

### 6. Advanced Types (`src/06-advanced-types.ts`)
Work with union types, intersection types, type guards, and utility types.

### 7. Modules (`src/07-modules.ts`)
Practice with ES6 modules and TypeScript module system.

## Instructions

- Complete each exercise file
- Make sure all code compiles without errors
- Follow TypeScript best practices
- Use strict typing (avoid `any` unless necessary)
- Add comments where helpful to explain your approach

## Time Limit

Recommended: 60-90 minutes

Good luck!
